--- id: 616 title: "Move over Slashdot, here's Cachedot!!!!" issue: 21 ---

As you might have noticed from the Slashdot link above, those clever peeps at Slashdot-land have noticed that their server is getting so popular these days, that at some times it's very hard to get on!!!!! So they've started up another server called Cachedot, which keeps a copy of all the articles in Slashdot, so if you can't get in Slashdot proper, then you can still see the latest news here!!!!!

http://cachedot.slashdot.org/
